# WalletConnect Guide - Connecting Wallets to dApps

## Overview

This browser uses **WalletConnect** to connect wallets to decentralized applications (dApps). No browser extensions needed!

## How It Works

WalletConnect is an open protocol that connects mobile wallets to desktop dApps via QR codes. It's the industry standard and works with all major dApps.

## Supported Wallets

### For Ethereum dApps (Uniswap, OpenSea, etc.):
- ✅ **MetaMask Mobile** (recommended)
- ✅ Any WalletConnect-compatible wallet

### For Solana dApps (Jupiter, Raydium, Magic Eden, etc.):
- ✅ **Phantom Mobile**
- ✅ **Solflare Mobile**
- ✅ Any WalletConnect-compatible wallet

## How to Connect

### Step 1: Install Mobile Wallet
- Download **MetaMask Mobile** from App Store/Play Store (for Ethereum)
- Download **Phantom Mobile** from App Store/Play Store (for Solana)

### Step 2: Visit a dApp
- Open Uniswap, Jupiter, or any dApp in this browser
- Click "Connect Wallet"

### Step 3: Select WalletConnect
- Choose "WalletConnect" from the connection options
- A QR code will appear

### Step 4: Scan QR Code
- Open your mobile wallet app
- Look for "WalletConnect" or "Scan QR" option
- Scan the QR code displayed in the browser

### Step 5: Approve Connection
- Approve the connection in your mobile wallet
- You're connected! Your wallet now works in the dApp.

## Benefits

✅ **No Browser Extensions** - Works without installing anything in the browser
✅ **Secure** - Your private keys stay in your mobile wallet
✅ **Universal** - Works with all major dApps
✅ **Mobile + Desktop** - Use your mobile wallet on desktop dApps

## Troubleshooting

**QR code not scanning?**
- Make sure your mobile wallet app supports WalletConnect
- Try refreshing the dApp page
- Ensure good lighting for QR code scanning

**Connection fails?**
- Make sure both devices are on the same network (or mobile data works)
- Try disconnecting and reconnecting
- Restart the dApp page
