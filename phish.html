üì° SITE CLONER

Purpose: Clone any website and host it locally to capture credentials.

Steps:

  1. Select a Template or enter a custom URL (e.g., https://example.com/login)
  2. Set Local Port (default: 8080) - this is where your cloned site will be hosted
  3. Set Redirect URL - where victims go after entering credentials (usually the real site)
  4. Click START SERVER
  5. Share the generated link (e.g., http://localhost:8080) in your phishing email
  6. Captured credentials appear in the LIVE FEED tab

üõ°Ô∏è Cloudflare Bypass

If the target site is protected by Cloudflare, the cloner includes automatic bypass:

  • User-Agent Spoofing: Mimics a real browser to avoid detection
  • Automatic Retry: Handles Cloudflare challenges automatically
  • Cookie Handling: Preserves session cookies for authenticated pages

‚úÖ Tip: If cloning fails, check the Server Log for details. Some sites have advanced anti-bot measures that may require manual intervention.

‚ö†Ô∏è Note: The cloned site runs on YOUR computer. To make it accessible to others, you'll need to use port forwarding or a tunneling service like ngrok.

Purpose: Send phishing emails using SMTP servers.

üé≠ URL MASKING

Make your phishing links look more legitimate by displaying one URL while linking to another:

  1. Enter the Display URL - what the victim sees (e.g., https://apple.com/verify)
  2. Enter the Actual URL - your phishing site (e.g., http://localhost:8080)
  3. Click "INSERT MASKED LINK INTO EMAIL"
  4. The masked link will be inserted into your email body at the cursor position

‚ö†Ô∏è Note: When the victim hovers over the link, their email client will show the actual URL in the status bar. This is a limitation of email security. The text will still show the display URL, which can be enough to fool less tech-savvy users.

üîê GMAIL SETUP

  1. Go to Google Account Security
  2. Enable 2-Step Verification (required)
  3. Go to App Passwords
  4. Generate a new App Password (16-character code like abcd efgh ijkl mnop)
  5. In Omega Phish:
    • SMTP Host: smtp.gmail.com
    • Port: 465 (SSL)
    • Username: Your Gmail address
    • Password: The 16-char App Password (NOT your regular password)

‚ö†Ô∏è Important: Gmail will overwrite the "FROM" address with your real email. True spoofing is not possible with Gmail.

üîê OUTLOOK SETUP

  1. Go to Microsoft Account Security

  3. Under App passwords, create a new one
  4. In Omega Phish:
    • SMTP Host: smtp.office365.com
    • Port: 587
    • Username: Your Outlook email
    • Password: The generated App Password

üí° TIPS & TROUBLESHOOTING

  • Email goes to spam? Use a realistic FROM address (not security@google.com). Use your actual Mailgun domain instead.
  • Connection timeout? Check firewall settings. Try port 465 (SSL) instead of 587.
  • Auth failed? Make sure you're using an App Password, not your regular password.
  • Email not received? Check spam folder. Check Mailgun logs (Sending ‚Üí Logs).
  • Cloned site looks broken? Some sites use anti-scraping measures. Try a simpler target first.
